# 📌 Project Overview

This project presents an interactive **Retail Store Analytics Dashboard** built using **Microsoft Power BI**.

The dashboard provides valuable insights into sales performance, profitability, customer behavior, product performance, payment methods, and city-wise business growth.

It enables decision-makers to monitor KPIs and identify trends for strategic business planning.
