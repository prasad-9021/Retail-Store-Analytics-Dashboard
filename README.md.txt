# 🛒 Retail Store Analytics Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Retail Store Analytics Dashboard developed in Microsoft Power BI.

The dashboard helps analyze retail sales performance, customer behavior, product categories, profitability, payment methods, and city-wise sales trends through interactive visualizations.

It enables business users to monitor KPIs and make data-driven decisions.

---

## 📊 Dashboard Highlights

- Total Sales Analysis
- Total Profit Analysis
- Total Transactions
- Quantity Sold
- Store-wise Sales
- Category-wise Performance
- City-wise Sales
- Monthly Sales Trend
- Monthly Profit Trend
- Customer Type Analysis
- Payment Mode Distribution
- Sales Funnel
- Matrix Report
- Interactive Filters (Store, City, Category, Product, Customer Type, Payment Mode)

---

## 🛠 Built Using

- Microsoft Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Data Visualization

---

## 📈 Business Insights

- Identified top-performing stores and cities.
- Compared sales across product categories.
- Analyzed monthly sales and profit trends.
- Studied customer purchasing behavior.
- Evaluated payment mode preferences.
- Built an interactive dashboard for business reporting.
